# Skills

A collection of reusable skill instructions organized by category.

## Adding a Skill

1. Create a category folder at the repository root.
2. Add your skill Markdown file directly inside that category folder.

Example:

```text
frontend/
├── nuxt-design-system.md
├── nuxt-frontend-design.md
└── nuxt-ui-review.md
```

Keep the structure simple: one category folder, then skill `.md` files inside it. Do not add an extra folder per skill.
